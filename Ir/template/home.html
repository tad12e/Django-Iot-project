Real-Time Environmental Condition Monitoring

Track, log, and visualize critical conditions like **temperature and humidity** from simulated $\text{IoT}$ devices using a robust $\text{Django}$ backend and $\text{REST}$ $\text{API}$ architecture.

Go to Live Dashboard Explore Device Setup

The Full-Stack IoT Pipeline

1. Embedded Firmware (C/C++)

Simulated $\text{Arduino}$ device reads sensor data (Temperature/Humidity), applies local threshold checks (LED alert), and formats the output into clean **JSON** payloads.

2. REST API & Data Ingestion

A secure $\text{Django}$ $\text{REST}$ $\text{Framework}$ ($\text{DRF}$) $\text{API}$ endpoint receives the $\text{HTTP}$ data, validates fields against rules, and persists the readings to the database.

3. Dashboard & Analysis

The $\text{Django}$ front-end fetches the latest data and uses $\text{Chart.js}$ to render line graphs for historical trend analysis and real-time status updates.
